# 🚀 Sunil Sahoo Portfolio

A modern and responsive developer portfolio built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**. This portfolio showcases projects, skills, experience, and contact information with smooth animations and a clean user experience.

## 🌐 Live Demo

```text
https://sunil-devfolio.netlify.app/
```

---

## ✨ Features

* Responsive design for all devices
* Modern UI with smooth animations
* Dark/Light theme support
* Interactive project showcase
* Skills section with technology icons
* Contact form integration using EmailJS
* Sticky social media links
* Scroll-to-top button
* SEO-friendly structure
* Fast performance with Vite

## ⚙️ Installation

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the URL displayed in your terminal.
```text 
http://localhost:5173/ 

```

Generate a production build:

## 📧 Contact Form Setup

This project uses EmailJS for sending emails directly from the contact form.

1. Create an EmailJS account.
2. Create a service.
3. Create an email template.
4. Add your Service ID, Template ID, and Public Key.
5. Configure them in the contact component.
